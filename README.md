# 📄 Professional Responsive CV Template

Welcome to my **Bootstrap-based CV** project! This project is a modern, clean, and fully responsive Resume/CV layout inspired by professional designs, built during my training at **ITI**.

---

## 🚀 Live Preview
You can view the live demo of this project here: [Click to View CV](https://hajar.github.io/Your-Repo-Name/)

---

## ✨ Features
* **Fully Responsive:** Optimized for Mobile, Tablet, and Desktop using **Bootstrap 5**.
* **Modern UI:** Clean card-based design with soft shadows and professional typography.
* **Custom Branding:** Custom "Teal" color theme (`#20c997`) applied across icons, progress bars, and footer.
* **Interactive Components:** * Dynamic **Progress Bars** for Skills and Languages.
    * Styled **Badges** for current job status.
    * **Bootstrap Icons** integration for a minimalist look.
* **No Grid Layout:** Built entirely using **Flexbox** and Bootstrap utilities as per project requirements.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling and teal theme overrides.
* **Bootstrap 5:** Core UI components.
* **Bootstrap Icons:** Scalable iconography.
